#Resumo dos Comandos Git:

Commit: 
Salva o estado atual do repositório com um identificador único (SHA1), formando um histórico de mudanças.

Branch: 
Cria uma linha paralela de desenvolvimento para novas funcionalidades, que pode ser mesclada com o ramo principal (master).

Tag: 
Marca uma versão específica do repositório de forma imutável, normalmente usada para lançamentos.

Clone: 
Copia todo o repositório (código e histórico) para um ambiente local.

Pull: 
Sincroniza as mudanças do repositório remoto para o local.

Push: 
Envia as mudanças do repositório local para o remoto.
